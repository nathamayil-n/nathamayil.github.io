Project Details

Car Comfort and Safety System

A simulation created using Node RED, MQTT and Thingsboard

A simulation where the system is capable of regulating car temperatures and windshields by analyzing the current weather conditions, while also tracking the safety of the owner through GPS

Node RED is a simulation tool, with various functionalities like the usage of webhooks, MQTT and other features. With the help of OpenWeather API, the weather data was obtained, analyzed and the car's temperature settings were changed with IoT automation using MQTT protocol. Thingsboard was also used to visualize these for the user to view them in a more presentable way. The system also tracks the car location and battery levels using GPS and sends messages to the emergency contacts during cases of low battery along with the user's location.

Technologies Used:

Node-RED Openweather OwnTracks Webhook Relays Thingsboard JavaScript

Project Details

Prediction of Air Pollution Deaths in Various Countries

Data Analysis and Visualization using Python and Tableau

The project was based on predicting the pollution levels with the help of existing data and the factors behind air pollution and its constituents.

This project focused on the study of air quality index between first world countries and the third world countries like the USA and India . There is a significant reduction in air quality around the world due to modernisation and more and more exploitation of the resources. The outcome was to provide a visualisation of the comparison and then use the algorithms like the Regression Analysis,LSTM and ARIMA models for the third world and the first world comparison and find the accuracy percentage.

Technologies Used:

Tableau Python

Project Details

Allergen Detection System

A Decision Tree based system developed using Python

This project was based on coming up with an accessible system that helps in analyzing and helping the users to identify related allergens and the food items containing it

The customer has to first create an account and answer a quiz that helps them identify the substances that they are allergic to. The customer will scan the barcode present in front of the food item. Scanning it will give all the details regarding the food item and whether it has any substances/compounds to which the person is allergic to and alert them accordingly. A DT model also heelps in identifying the frequency of the symptoms caused by these allergens.
